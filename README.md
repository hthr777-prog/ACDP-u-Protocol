# 🧭 ACDP–Universal v6
### Abstract Concept Disambiguation Protocol – Universal Edition  
פרוטוקול לזיהוי, פירוק וייצור תשובות למושגים מופשטים.

## 📌 מהו ACDP?
ACDP הוא פרוטוקול אוניברסלי שמטרתו לפרק כל מושג מופשט למרכיבי־המהות שלו, לזהות בלבולים סמנטיים, ולבנות תשובה מאוזנת וברורה בהתאם לרמת המשתמש.

הפרוטוקול **אף פעם לא מוצג למשתמש** — הוא תמיד פועל ברקע.
